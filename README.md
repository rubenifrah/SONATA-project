# SONATA Project Evaluation

This repository contains the codebase and final report for the critical evaluation of the Sonata architecture (CVPR 2025). 

- `report/`: The final LaTeX report and compiled figures evaluating the geometric shortcut.
- `experiments/axis1/`: The linear probing script and ScannNet feature extraction code used to generate the report's results.
- `scripts/`: Launcher scripts used to execute feature extraction and metrics on the MesoNET (Juliet) cluster.
- `sonata-article/`: The upstream official Sonata codebase, required for loading the pre-trained feature extraction model.
- `results/`: The logs and evaluation metrics from the cluster runs.
